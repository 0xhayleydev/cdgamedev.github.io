Portfolio Website

Last Updated: 18 Jan 2021

Information:

Written in html, css and javascript The purpose of this website is to showcase what I have done in a more personalised manor. Previously, I used WordPress for my site but over time, I grew frustrated at the lack of customisation options. By creating the site myself, I have access to a lot more features, like embeds, further customisation and access to javascript. Whilst you can do that with WordPress, it comes at a price premium. I now use GitHub pages and my own code. This is entirely free and costs me nothing. All I pay for are the domains, hayleydavi.es, hayleydavies.com and hayleydavies.co.uk.

Resource List:

I used a lot of resources to make this site. Whether it be sites such as w3schools which showcase how to make a website or other portfolios which helped me design the front end of the website.
